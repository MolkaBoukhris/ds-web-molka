Matiéres Description
SE Un système d'exploitation est un logiciel qui gère les ressources matérielles et logicielles d'un ordinateur, tout en offrant une interface utilisateur et une plateforme pour exécuter des applications (ex. : Windows, Linux, macOS).
Base de données Une base de données est un système organisé permettant de stocker, gérer et récupérer efficacement des informations structurées.
Probabilité Les probabilités sont une branche des mathématiques qui étudie les chances qu'un événement se produise, en modélisant l'incertitude et en calculant les résultats possibles dans des situations aléatoires.
Language C Le langage C est un langage de programmation généraliste, bas niveau et performant, utilisé pour le développement de systèmes, logiciels embarqués, et applications nécessitant une grande efficacité.
